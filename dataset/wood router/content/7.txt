Fixed-Base Routers












2.3 HP Electronic Fixed-Base Router




Trigger-control system – trigger power switch located in handle for enhanced control


Powerful 15 Amp motor – produces 10,000 to 25,000 RPM and 2.3 horsepower


Precision-centering design – makes it easy to keep bit on cutline


Versatile fixed-base depth adjustment – features 1-5/8 In. continuous microfine adjustment range


Above-table depth adjustment when fixed base is installed in router table


Constant Response circuitry – maintains constant speed under load


Always-on LED lighting – for enhanced visibility in bit area


Soft-start function – reduces start-up torque


Two-stage quick clamp system – for fast tool-free conversion


Variable-speed dial – for matching speed to workpiece and task


Contoured soft grip handles – for enhanced control and comfort


Fixed base has threaded holes for mounting to router table positioned in common three-hole pattern


Cord swivel – for added mobility


Compatible with optional RA1129 Template Guide Adapter Set – for easy change of Bosch quick-release template guides and common threaded template guides

MRF23EVS Professional: Further information


Product Highlights


The Bosch MRF23EVS 2.3 HP Electronic Fixed-Base Router delivers optimum user control, with the trigger-controlled power system that features a power switch in the fixed-base handle. This router has the power to get results, with its 15 Amp motor and adjustable speed settings. The variable-speed dial allows the operator to set the optimal speed. The Bosch Constant Response circuitry in the motor maintains a constant speed throughout the cut, resulting in clean, accurate product. The router has microfine bit-depth adjustment, with a continuous depth range of 1-5/8 in., allowing for accurate and fast changes, while its precision-centering design allows the bit to be centered with the sub-base or optional template guides. Users can glide along the intended cutline using dovetail fixtures, templates, jigs and other guidance devices.
