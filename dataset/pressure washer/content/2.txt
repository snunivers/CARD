Features






Standard equipment




















ROTATING SPRAY LANCE


FLEXIBLE WORKING. Thanks to the rotating spray lance, you can easily adapt the alignment of the high-pressure jet to the work situation. The high-pressure lance is ergonomically secure in the hand at all times.


















Rotary nozzle


Powerful rotary nozzle for tackling stubborn dirt on hard-wearing surfaces. The rotary motion combines the cleaning power of a concentrated pencil jet with the capacity to clean large areas. 


















Cleaning agent spray set 


The spray set delivered with the tool makes it possible to add cleaning agent to the spray jet in order to tackle stubborn dirt. The spray set is also easy to attach to the spray lance.


















Storing accessories on the tool


The high-pressure trigger, spray lance, fan jet nozzle, rotary nozzle, high-pressure hose and cleaning agent spray set can be stored securely.


















Fan jet nozzle, adjustable pressure


Fan jet nozzle with pressure regulation to clean different types of surfaces quickly and effectively.


















Cable hook


The two cable hooks allow the cable to be wound up and neatly and securely stored on the tool.


















Integrated wheel kit 


The integrated wheel kit allows the high-pressure cleaner to be manoeuvred comfortably. 


















Anti-drill and quick-release coupling


The anti-drill and quick-release coupling makes setting up the high-pressure cleaner easier and prevents the high-pressure hose from tangling while you work, allowing the full length of the hose to be used.


















Park position


Allows you to switch off the assembled high-pressure lance when taking breaks, and to store the lance on the tool.
