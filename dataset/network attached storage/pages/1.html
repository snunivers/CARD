                                  FUJITSU ETERNUS SF V16.9
                                  Storage Management Software

                                  ETERNUS SF storage management software supports overall daily operations within the entire ETERNUS AF and ETERNUS DX storage infrastructure by offering an easy-to-use graphical user interface for all management tasks,like health monitoring, capacity management, provisioning, local and remote replication and disaster recovery as well as the management of unified storage. At the same time it reduces significantly administrator’s training efforts with regards to cost and time.

                                  Fujitsu Storage ETERNUS Multipath Driver
                                  (only available for APAC region)

                                  ETERNUS Multipath Driver software achieves high-reliability and performance systems due to multipath connection between storages and servers. In case of path failure ETERNUS Mutpath Driver switches the access path to improves system reliability. In addition, further improved system performance is ensured by the load balancing function between multiple paths.
